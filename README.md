# BabyGuard 宝贝护卫队

一款以育儿为主题的塔防策略游戏，适合儿童和家长一起游玩。

## 游戏简介

保护宝宝的小房间免受"睡眠恶魔"、"挑食怪兽"、"屏幕幽灵"等可爱敌人的攻击！玩家扮演"家长团"成员，建造各种特色的防御塔来击退入侵的怪物。

## 游戏特色

- **5种特色防御塔**：妈妈塔（治疗）、爸爸塔（减速）、奶奶塔（眩晕）、医生塔（高伤害）、厨师塔（范围攻击）
- **4种敌人类型**：发脾气怪、睡眠兽、蔬菜讨厌鬼、屏幕幽灵
- **10个波次挑战**：难度递增，最终波为BOSS战
- **儿童友好设计**：可爱的画风、明亮的色彩、简单的操作

## 玩法说明

1. 点击底部选择要放置的防御塔
2. 在绿色高亮区域点击放置防御塔
3. 点击"开始波次"按钮启动敌人进攻
4. 击败敌人获得金币，用于建造更多防御塔
5. 阻止敌人到达宝宝房间，保护20条生命

## 操作方式

| 操作 | 按键 |
|------|------|
| 放置防御塔 | 鼠标左键 |
| 取消选择 | 右键 / ESC |
| 开始波次 | 空格键 / 点击按钮 |

## 防御塔一览

| 塔名 | 费用 | 功能 |
|------|------|------|
| 妈妈塔 | 100 | 治疗范围内所有塔，每秒恢复5点HP |
| 爸爸塔 | 100 | 减速范围内所有敌人40%速度 |
| 奶奶塔 | 150 | 眩晕范围内随机敌人1.5秒 |
| 医生塔 | 200 | 对最远敌人造成高额单体伤害 |
| 厨师塔 | 175 | 对范围内所有敌人造成范围伤害 |

## 敌人类型

| 敌人 | HP | 特性 |
|------|-----|------|
| 发脾气怪 | 60 | 移动速度快 |
| 睡眠兽 | 100 | 周期性睡觉（睡觉时无敌） |
| 蔬菜讨厌鬼 | 200 | 高血量坦克 |
| 屏幕幽灵 | 120 | 免疫减速效果 |
